Fonts and texts

You can alter text boldness through strong tag or italicise using em tag. You can create teletype or type-writer style text with the teletype tag. The citation tag will give citation impression eg. italicise.

You can alter font using the font size tag, or colour using the font color tag.



Images

You can include an image using the img tag, which will include the src="" property to show image source and the alt="" property to show image name if image doesn't appear

You can alter the width and height, using the style attribute or the width and height attributes.

HTML allows animated gifs through image tags also.

An image can be a link if you put the image tag inside the anchor tag.

These are two examples of attaching images. The first is an image which is saved in the same folder as the HTML, the second is an image saved in another folder so you need dash, folder name, file name.
